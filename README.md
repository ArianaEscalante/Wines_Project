# Wines_Project
- Performed end-to-end data analysis on a wine dataset using Python.
- Cleaned and transformed raw data using pandas.
- Developed a “value for money” metric.
- Conducted exploratory analysis to identify top-performing wines and regions.
- Visualized insights using matplotlib and seaborn and provided data-driven recommendations.
